#YouTube Data Analysis Project 
This project focuses on building an end-to-end data pipeline for analyzing YouTube video data. It implements key stages of the data engineering lifecycle, including data ingestion, transformation, storage, and visualization. By integrating the YouTube API, the pipeline collects and processes video performance metrics, such as views, likes, and engagement, to uncover valuable insights. The project emphasizes scalable and efficient data handling practices to ensure smooth analysis and reporting, showcasing essential concepts like API integration, ETL workflows, and data visualization.
